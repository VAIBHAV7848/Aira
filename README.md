A local-first, autonomous AI coding agent.

## Quick Start

1. **Activate Environment**
   ```powershell
   .\.venv\Scripts\activate
   ```

2. **Run Aira**
   ```powershell
   python -m aira.main
   ```

3. **Talk to Aira**
   Open Telegram and search for your bot. Send `/start`.

## Requirements
- Python 3.11+
- Ollama running locally (`qwen2.5:7b-instruct`)
- Telegram Bot Token in `.env`

## Commands
- `/start` - Wake up Aira
- `/status` - Check current task & budget
- `/cancel` - Stop current task
